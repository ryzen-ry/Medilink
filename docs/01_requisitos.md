#Hito 1 (Semana 5 - APF1): Fundamentos, Arquitectura y Gobierno de Código 
En esta fase, el equipo establece los cimientos del proyecto y las reglas estrictas de 
colaboración. 
• Repositorio y Documentación: Creación del repositorio y redacción de la 
primera mitad del README.md (hasta Metodología). Configuración del 
.gitignore para Java/Spring Boot. 
• Estructura Base: Subida del cascarón de Spring Boot (controladores vacíos, 
configuración de base de datos H2 o MySQL) a la rama develop. 
• Nivel Avanzado (Gobierno de Código):  
o Protección de Ramas: Los estudiantes deben configurar reglas en 
GitHub para que la rama main y develop estén protegidas, exigiendo al 
menos un revisor (Code Review) antes de aceptar un Pull Request. 
o Conventional Commits: Uso obligatorio de prefijos estándar en la 
terminal (ej. feat: agrega entidad producto, docs: actualiza 
readme).
